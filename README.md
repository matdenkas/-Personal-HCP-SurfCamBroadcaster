# HCP-Surfcam-Webserver


## Description
A node/react webserver used to brodcast the surfcam at HCP  Horace Calwell Pier (HCP). 


## Getting Started

### Dependencies
* npm i express

### Installing

* Clone and leave directories as is.

### Executing program

#### Server
```
npm start
```

#### Client
```
cd client
npm start
```
